# @qnsp/browser-sdk

## 0.1.3

### Patch Changes

- Initial tracked CHANGELOG entry. Earlier 0.1.0–0.1.2 releases shipped the
  browser-side SDK surface for QNSP public APIs: typed fetch wrappers,
  session helpers, WebAuthn/FIDO2 registration and assertion flows, and
  browser-safe PQC primitives where available.
- All subsequent releases will be driven by Changesets. See
  https://docs.qnsp.cuilabs.io/sdk/browser-sdk for usage, supported
  browsers, and policy for handling PQC key material in the DOM.
