# @qnsp/mcp-server

Official QNSP MCP server for AI assistants.

## Install

```bash
pnpm add -g @qnsp/mcp-server
```

## Usage

```bash
export QNSP_API_KEY="qnsp_pqc_api_..."
qnsp-mcp
```

The server activates against QNSP with `@qnsp/sdk-activation` and exposes tenant-scoped tools for vault, KMS, crypto posture, audit, billing, and platform health.

Docs: https://docs.qnsp.cuilabs.io/sdk/mcp-server
